# 🍳 실리트 압력밥솥 AS 견적서 관리자

실리트 압력밥솥 AS 부품 견적을 내고 전문적인 견적서를 출력할 수 있는 스마트 도구입니다.

## 주요 기능
- **✨ 프리미엄 디자인**: 현대적이고 깔끔한 UI로 고객에게 신뢰감을 줍니다.
- **💾 자동 저장**: 작성 중인 데이터가 브라우저에 자동으로 저장되어 안심하고 사용할 수 있습니다.
- **📥 백업 및 복원**: JSON 파일을 통해 데이터를 백업하고 다른 기기에서 복원할 수 있습니다.
- **🖼️ 부품 가이드 이미지**: 참고용 부품 명칭 이미지를 업로드하여 상시 확인할 수 있습니다.
- **🖨️ 최적화된 출력**: 불필요한 요소를 제외하고 전문적인 견적서 형식으로 출력됩니다.

## 배포 방법 (Deployment)

이 프로젝트는 Vercel에 최적화되어 있습니다.

1. **GitHub 연동**: 프로젝트를 GitHub에 업로드합니다.
2. **Vercel 연결**: [Vercel](https://vercel.com)에서 새로운 프로젝트를 생성하고 해당 레포지토리를 연결합니다.
3. **환경 변수**: `GEMINI_API_KEY`가 필요한 경우 Vercel 환경 변수 설정에서 추가하세요.
4. **빌드 설정**:
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Framework Preset: `Vite`

## 데이터 관리
- 우측 상단의 **'백업'** 버튼을 누르면 현재 데이터가 `.json` 파일로 저장됩니다.
- **'복원'** 버튼을 통해 이전 백업 파일을 다시 불러올 수 있습니다.
